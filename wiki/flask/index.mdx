# Flask

> [Flask](https://flask.palletsprojects.com/en/2.3.x/)는 **가벼운 [WSGI](https://wsgi.readthedocs.io/) 웹 애플리케이션 프레임워크**입니다. 시작하기 쉽고 빠르게 만들 수 있도록 설계되었으며 복잡한 애플리케이션으로 확장할 수 있습니다. 원래는 [Werkzeug](https://werkzeug.palletsprojects.com/)와 [Jinja](https://jinja.palletsprojects.com/)를 감싸는 간단한 래퍼로 시작했지만, 이제는 가장 인기 있는 Python 웹 애플리케이션 프레임워크 중 하나입니다.  
> Flask는 의존성이나 프로젝트 레이아웃을 강제하지 않고 제안을 제공합니다. 개발자가 사용할 도구와 라이브러리를 선택하는 것은 개발자의 몫입니다. 커뮤니티에서 제공하는 많은 확장 기능을 사용하면 새로운 기능을 쉽게 추가할 수 있습니다.
