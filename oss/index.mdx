# 📌 오픈소스로 배우기

요즘 들어서 코드를 짜는데 망설임을 느끼게 되었다.

> _"글로벌 변수를 넣는 게 맞을까?"_  
> _"클래스로 가져갈까? 함수로 가져갈까?"_  
> _"상수는 별도 파일에서 관리할까? 아니면 여기서만 쓰는 거니 여기다 선언할까?"_  
> ...

이런 궁금증에 대한 답변은 주로 사내 코드에서 얻고 있는데, 확실히 좋은 코드 하나 보고 따라해보는게 큰 도움이 되었다.

하지만 이제 사내 코드로는 한계를 느껴 어느 정도 검증이 완료된 오픈소스로 **"코드 잘 짜는 법"**을 배워보려 한다.
