# Apache Hadoop

> [Apache Hadoop](https://hadoop.apache.org/) 소프트웨어 라이브러리는 **단순한 프로그래밍 모델을 사용하여 컴퓨터 클러스터간에 대규모 데이터 세트를 분산 처리할 수 있도록 하는 프레임워크**입니다. 이는 단일 서버에서 수천 대의 컴퓨터까지 확장 가능하며, 각각의 컴퓨터는 로컬 계산 및 저장을 제공합니다. 높은 가용성을 제공하기 위해 하드웨어에 의존하는 것이 아니라 라이브러리 자체가 어플리케이션 레이어에서 장애를 감지하고 처리하도록 설계되어 있으므로, 각각 장애가 발생할 가능성이 있는 컴퓨터 클러스터 위에서 높은 가용성의 서비스를 제공합니다.
