# 🦖 Docusaurus

> 도큐사우루스는 정적 사이트 생성 도구입니다. 클라이언트에서 빠른 탐색 기능을 지원하고 사이트를 좀 더 인터랙티브하게 만들 수 있게 React의 기능을 최대한 끌어올린 단일 페이지 애플리케이션을 만들 수 있습니다. 문서 기능은 설치 후 바로 사용할 수 있으며 원하는 어떤 종류의 사이트(개인 웹 사이트, 제품 블로그, 마케팅 랜딩 페이지 등)든지 만들 수 있습니다.
>
> 🔗 [개요 | Docusaurus](https://docusaurus.io/ko/docs/next)

## 커스텀을 위한 험난한 여정

### 날짜 포맷

날짜 포맷은 `MM/DD/YYYY` 포맷이다.

### Algolia 크롤링 이슈

Algolia에서 크롤링이 안 되고 브라우저에 URL 입력 후 화면에 404가 뜬 다음 리다이렉트된다면, trailingSlash 이슈일 수 있다.
trailingSlash 관련해서는 [URL 끝에 '/'는 왜 붙이는 걸까?](https://djkeh.github.io/articles/Why-do-we-put-slash-at-the-end-of-URL-kor/)를 참고!

### 사이드바 커스텀

사이드바에서 `category` 타입인 경우, index 문서를 만들고 싶다면 해당 폴더 말고 최상위 폴더인 `/doc`에 동일 id로 파일 만들고 `link`에 추가하면 된다.

```javascript
{
  type: "category",
  label: "🐋 Docker",
  link: {
    type: "doc",
    id: "docker", // 생성한 index 문서의 id 값
  },
  items: [
    {
      type: "autogenerated",
      dirName: "software-engineering/docker",
    },
  ],
}
```

사이드바에 문서 말고 다른 걸 넣고 싶다면, `html` 타입을 활용하면 좋다. 구분선을 추가하고 싶은 경우 아래처럼 `<hr/>` 태그를 추가해주면 된다.

```javascript
{
  type: "html",
  value: '<hr style="margin: 12px 0;" />',
}
```
