# ⚗️ SQLAlchemy

> SQLAlchemy는 애플리케이션 개발자에게 SQL의 모든 기능과 유연성을 제공하는 Python SQL 도구 키트 및 객체 관계형 매퍼입니다. 효율적이고 고성능 데이터베이스 액세스를 위해 설계된 잘 알려진 엔터프라이즈급 지속성 패턴의 전체 제품군을 간단한 Python 도메인 언어에 맞게 조정하여 제공합니다.
>
> 🔗 [SQLAlchemy Documentation](https://www.sqlalchemy.org/)
