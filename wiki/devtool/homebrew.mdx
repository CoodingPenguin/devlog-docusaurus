# Homebrew 명령어

## 설치

- xcode command line tools 설치

```bash
xcode-select --install
```

- homebrew 설치 (Intel)

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

- homebrew 설치 (Apple Silicon/M1)

```bash
# arm64
arch -arm64 /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

# x86_64
arch -x86_64 /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
```

## 명령어

### brew를 최신 버전으로 업데이트

```bash
brew update
```

### 패키지 업그레이드

`brew update`가 선행된다.

```bash
# 전체 패키지 업그레이드
brew upgrade

# 인자로 넘긴 패키지만 업그레이드
brew upgrade <패키지명1> <패키지명2> <패키지명3>
```

### 업그레이드가 필요한 패키지 확인

```bash
brew oudated
```

### 패키지 검색

```bash
brew search <패키지명>
```

### 패키지 설치

```bash
brew install <패키지명>[@버전]
```

### 설치된 패키지 확인

```bash
brew list | ls

# 더 간단히
brew leaves
```

### 패키지 정보 확인

```bash
brew info <패키지명>
```

### 패키지 삭제

```bash
brew uninstall | remove | rm <패키지명>
```

### 최신 버전 외 버전의 패키지 삭제

```bash
brew cleanup <패키지명>
```

### 패키지 종속성 확인

```bash
brew deps --tree --installed <패키지명>
```

### 문제 진단

```bash
brew doctor
```
