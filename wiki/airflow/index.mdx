# Apache Airflow

> [Apache Airflow](https://github.com/apache/airflow)는 배치 지향 워크플로우를 개발, 스케줄링 및 모니터링하는 오픈소스 플랫폼입니다. Airflow의 확장 가능한 Python 프레임워크를 사용하여 거의 모든 기술과 연결된 워크플로우를 구축할 수 있습니다. 웹 인터페이스를 사용하여 워크플로우의 상태를 관리할 수 있습니다. Airflow는 노트북의 단일 프로세스에서부터 분산 설정까지 다양한 방식으로 배포할 수 있으며, 가장 큰 워크플로우도 지원합니다.
