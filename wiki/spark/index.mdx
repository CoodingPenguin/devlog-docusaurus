# Apache Spark

> Apache Spark는 **대규모 데이터 처리를 위한 통합 분석 엔진**입니다. Java, Scala, Python 및 R에서 고수준 API를 제공하며, 일반 실행 그래프를 지원하는 최적화된 엔진을 제공합니다. 또한 SQL 및 구조화된 데이터 처리를 위한 [Spark SQL](https://spark.apache.org/docs/latest/sql-programming-guide.html), pandas 작업 부하를 위한 [pandas API on Spark](https://spark.apache.org/docs/latest/api/python/getting_started/quickstart_ps.html), 기계 학습을 위한 [MLlib](https://spark.apache.org/docs/latest/ml-guide.html), 그래프 처리를 위한 [GraphX](https://spark.apache.org/docs/latest/graphx-programming-guide.html), 그리고 점진적 계산 및 스트림 처리를 위한 [구조화된 스트리밍](https://spark.apache.org/docs/latest/structured-streaming-programming-guide.html)과 같은 고수준 도구의 다양한 기능도 지원합니다.
